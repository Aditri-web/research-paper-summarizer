
# Research Paper Summarizer 📄🤖

A deep learning project focused on fine-tuning Large Language 
Models (LLMs) to generate concise, accurate summaries of academic 
research papers.

## Overview
This project fine-tunes a pre-trained transformer model on the 
CNN/DailyMail dataset to learn summarization, then applies it 
to summarize research papers automatically.

## Pipeline
Data Collection → EDA → Preprocessing → 
Fine-tuning → Evaluation → Deployment

## Tech Stack
- Python 3.x
- HuggingFace Transformers
- PyTorch
- CNN/DailyMail Dataset
- Jupyter Notebooks
- ROUGE Evaluation Metrics
